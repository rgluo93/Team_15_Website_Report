Weeks 11-12: 26/01 - 08/02

We developed major features including the WhatsApp client and initial translation integration, and began hardening deployment and test webhooks for reliable operation.

<!--truncate-->

Towards the end of January is where our development really took off. Previously, we assigned tasks to each member, and the repo saw large changes regarding the initialization of features like WhatsApp and translation. This week, we created the WhatsApp client, which calls on the WhatsApp Cloud API and provides methods to send text messages, then created the FastAPI endpoint for access. We also set up a test webhook using Render, which importantly allows the chatbot to receive messages, so that it can mark messages as read and respond appropriately. Initially, we were using a personal Meta Developer account to set up a test app, allowing access to the API with a personal token and phone number ID. Regarding the translation service, we integrated Meta’s NLLB-200 model for English↔Swahili translation, loading the model and tokenizer locally via Hugging Face Transformers.
