Weeks 7-8: 01/12 - 14/12

Built core architecture diagrams and delivered first working prototypes for internal testing.

<!--truncate-->

During this period, we focused on developing the AI chat service. We created the core LLM service, which uses LangChain to construct prompts and calls Azure's OpenAI to get AI responses. The first few features, generate user summary and scenario-based learning feedback were also successfully implemented, which use the chat service with tailored system prompts in order to generate the appropriate response. Finally, we made all these features accessible through FastAPI endpoints. Since the Dignitas team were facing issues with providing us access the frontend, we made an interactive demo from the command line to demonstrate our progress in our regular meeting.
