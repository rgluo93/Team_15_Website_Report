## Scenario-Based Learning Feedback

After a teacher submits their response to a scenario question, an LLM compares it against the expert answer and generates personalised feedback covering strengths, areas for improvement and actionable recommendations.

## AI Chatbot (Teacher)

Teachers can ask the AI Assistant questions about module content at any time. The chatbot provides contextual answers drawn from the learning material, supporting teachers when they are confused or need clarification.

## Education Officer AI Agent

Education officers interact with a dedicated AI Agent that can query live data, identify underperforming teachers and take actions such as composing and sending WhatsApp reminders — all from a single chat interface.
